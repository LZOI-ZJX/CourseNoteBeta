# 偏微分方程差分方法习题与解答整理

## 1. 用五点差分格式求解 Poisson 方程的边值问题

**题目：**
求解 Poisson 方程的边值问题：
$$
\begin{cases}
\Delta u = 16, & (x,y) \in D, \\
u = 0, & (x,y) \in \partial D,
\end{cases}
$$
其中 $D = \{(x,y) \mid |x|, |y| < 1\}$。
（1）用正方形网格（$k=h$）列出相应的差分方程；
（2）对 $h=1, h=\frac{1}{2}$ 分别求解。

**解答：**

### （1）用正方形网格列出相应的差分方程
在内部节点上，使用二阶精度的中心差分格式来近似二阶偏导数，代入 $\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 16$ 可得五点差分格式：
$$\frac{u_{i+1,j} + u_{i-1,j} + u_{i,j+1} + u_{i,j-1} - 4u_{i,j}}{h^2} = 16$$
整理后，相应的差分方程为：
$$u_{i+1,j} + u_{i-1,j} + u_{i,j+1} + u_{i,j-1} - 4u_{i,j} = 16h^2$$

### （2）分别对 $h=1$ 和 $h=\frac{1}{2}$ 求解

**情形一：当 $h=1$ 时**
区域内只有原点 $(0,0)$ 这唯一一个内部节点。其四个相邻节点均落在边界 $\partial D$ 上，值为 $0$。
代入差分方程：
$$0 + 0 + 0 + 0 - 4u(0,0) = 16 \times 1^2 \implies -4u(0,0) = 16 \implies u(0,0) = -4$$
**结论：** 内部节点的解为 $u(0,0) = -4$。

**情形二：当 $h=\frac{1}{2}$ 时**
区域内部共有 9 个节点。利用对称性，可将未知节点简化为 3 组：中心点 $u_1(0,0)$、四个坐标轴点 $u_2$、四个对角线角点 $u_3$。方程右端项为 $16 \times (\frac{1}{2})^2 = 4$。
分别列出差分方程：
1. 对于中心点 $u_1$：$4u_2 - 4u_1 = 4 \implies u_2 - u_1 = 1$
2. 对于坐标轴点 $u_2$：$u_1 + 2u_3 - 4u_2 = 4$
3. 对于对角线角点 $u_3$：$2u_2 - 4u_3 = 4 \implies u_2 - 2u_3 = 2$
联立求解得：
$u_2 = -3.5$
$u_1 = -4.5$
$u_3 = -2.75$
**结论：** 中心点解为 $u(0,0) = -4.5$；坐标轴点解为 $u(\pm 0.5, 0) = u(0, \pm 0.5) = -3.5$；对角线角点解为 $u(\pm 0.5, \pm 0.5) = -2.75$。

## 2. 列出用五点差分格式求解 Laplace 方程的第三边值问题

**题目：**
列出用五点差分格式求解 Laplace 方程的第三边值问题（$h=k=\frac{1}{2}$）的线性代数方程组：
$$
\begin{cases}
\Delta u = 0, & 0 < x,y < 1, \\
u_x - u = 1 + y, & x = 0, 0 \leqslant y \leqslant 1, \\
u_x + u = 2 - y, & x = 1, 0 \leqslant y \leqslant 1, \\
u_y - u = -1 - x, & y = 0, 0 \leqslant x \leqslant 1, \\
u_y + u = -2 + x, & y = 1, 0 \leqslant x \leqslant 1
\end{cases}
$$

**解答：**

**1. 边界条件的离散化（虚节点法）**
利用中心差分离散边界一阶导数（注意 $2h = 1$）：
* 左边界：$u_{-1,j} = u_{1,j} - u_{0,j} - 1 - y_j$
* 右边界：$u_{3,j} = u_{1,j} - u_{2,j} + 2 - y_j$
* 下边界：$u_{i,-1} = u_{i,1} - u_{i,0} + 1 + x_i$
* 上边界：$u_{i,3} = u_{i,1} - u_{i,2} - 2 + x_i$

**2. 各节点差分方程的推导**
将网格点记为 $u_{i,j}$ ($i,j \in \{0,1,2\}$)，代入内部控制方程 $u_{i-1,j} + u_{i+1,j} + u_{i,j-1} + u_{i,j+1} - 4u_{i,j} = 0$ 中：
* **内部节点 $(1,1)$**：$u_{0,1} + u_{2,1} + u_{1,0} + u_{1,2} - 4u_{1,1} = 0$
* **下边界中心 $(1,0)$**：$u_{0,0} - 5u_{1,0} + u_{2,0} + 2u_{1,1} = -1.5$
* **左边界中心 $(0,1)$**：$u_{0,0} - 5u_{0,1} + u_{0,2} + 2u_{1,1} = 1.5$
* **右边界中心 $(2,1)$**：$u_{2,0} - 5u_{2,1} + u_{2,2} + 2u_{1,1} = -1.5$
* **上边界中心 $(1,2)$**：$u_{0,2} - 5u_{1,2} + u_{2,2} + 2u_{1,1} = 1.5$
* **左下角 $(0,0)$**：$-3u_{0,0} + u_{1,0} + u_{0,1} = 0$
* **右下角 $(2,0)$**：$-3u_{2,0} + u_{1,0} + u_{2,1} = -2$
* **左上角 $(0,2)$**：$-3u_{0,2} + u_{0,1} + u_{1,2} = 2$
* **右上角 $(2,2)$**：$-3u_{2,2} + u_{2,1} + u_{1,2} = 0$

**3. 最终的线性代数方程组**
$$
\begin{cases}
-3u_{0,0} + u_{1,0} + u_{0,1} = 0 \\
u_{0,0} - 5u_{1,0} + u_{2,0} + 2u_{1,1} = -1.5 \\
u_{1,0} - 3u_{2,0} + u_{2,1} = -2 \\
u_{0,0} - 5u_{0,1} + u_{0,2} + 2u_{1,1} = 1.5 \\
u_{1,0} + u_{0,1} - 4u_{1,1} + u_{2,1} + u_{1,2} = 0 \\
u_{2,0} + 2u_{1,1} - 5u_{2,1} + u_{2,2} = -1.5 \\
u_{0,1} - 3u_{0,2} + u_{1,2} = 2 \\
2u_{1,1} + u_{0,2} - 5u_{1,2} + u_{2,2} = 1.5 \\
u_{2,1} + u_{1,2} - 3u_{2,2} = 0
\end{cases}
$$

## 3. 试求出解 Poisson 方程的差分格式的截断误差

**题目：**
试求出解 Poisson 方程 $-\Delta u = f(x,y)$ 的差分格式：
$-(u_{i+1,j+1} + u_{i+1,j-1} + u_{i-1,j+1} + u_{i-1,j-1} - 4u_{ij}) = 2h^2 f_{ij}$
的截断误差。

**解答：**

**1. 定义截断误差**
截断误差 $T_{ij}$ 定义为：
$$T_{ij} = -\frac{1}{2h^2} \left[ u_{i+1,j+1} + u_{i+1,j-1} + u_{i-1,j+1} + u_{i-1,j-1} - 4u_{ij} \right] + (u_{xx} + u_{yy})_{ij}$$

**2. 二元 Taylor 展开**
将交叉点 $u(x \pm h, y \pm h)$ 在节点处进行展开：
$$u(x \pm h, y \pm h) = u \pm h u_x \pm h u_y + \frac{1}{2!} (h^2 u_{xx} \pm 2h^2 u_{xy} + h^2 u_{yy}) + \mathcal{O}(h^3) + \frac{h^4}{24} (u_{xxxx} \pm 4u_{xxxy} + 6u_{xxyy} \pm 4u_{xyyy} + u_{yyyy}) + \mathcal{O}(h^5)$$

**3. 合并同类项**
将四个交叉方向的点相加，包含 $h$ 奇数次方的项以及交叉偏导数的奇数次项将全部抵消：
$$u_{i+1,j+1} + u_{i+1,j-1} + u_{i-1,j+1} + u_{i-1,j-1} = 4u + 2h^2(u_{xx} + u_{yy}) + \frac{h^4}{6}(u_{xxxx} + 6u_{xxyy} + u_{yyyy}) + \mathcal{O}(h^6)$$

**4. 计算最终截断误差**
代回截断误差的定义式中：
$$
\begin{aligned}
T_{ij} &= -\frac{1}{2h^2} \left[ 2h^2(u_{xx} + u_{yy}) + \frac{h^4}{6}(u_{xxxx} + 6u_{xxyy} + u_{yyyy}) + \mathcal{O}(h^6) \right] + (u_{xx} + u_{yy}) \\
&= -\frac{h^2}{12}(u_{xxxx} + 6u_{xxyy} + u_{yyyy}) + \mathcal{O}(h^4)
\end{aligned}
$$

**结论：**
该差分格式的截断误差为：
$$T_{ij} = -\frac{h^2}{12}(u_{xxxx} + 6u_{xxyy} + u_{yyyy})_{ij} + \mathcal{O}(h^4)$$
具有二阶空间精度。
