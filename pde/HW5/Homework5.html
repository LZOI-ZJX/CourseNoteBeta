Prob.1

1. 加权隐式格式定义

对于一维波动方程 \(\frac{\partial^2 u}{\partial t^2} = a^2 \frac{\partial^2 u}{\partial x^2}\)，考虑如下加权隐式差分格式：

\[ \frac{u_j^{n+1} - 2u_j^n + u_j^{n-1}}{\tau^2} = a^2 \left[ \theta \delta_x^2 u_j^{n-1} + (1-2\theta) \delta_x^2 u_j^n + \theta \delta_x^2 u_j^{n+1} \right] \]

其中 \(\delta_x^2\) 为二阶中心差分算子： \[\delta_x^2 u_j^n = \frac{u_{j+1}^n - 2u_j^n + u_{j-1}^n}{h^2}\]

2. 泰勒展开推导

为了验证截断误差 \(R_{ij}(h, \tau)\)，我们在点 \((x_j, t_n)\) 处对各算子进行展开。

2.1 时间项展开

\[ \frac{u_j^{n+1} - 2u_j^n + u_j^{n-1}}{\tau^2} = \frac{\partial^2 u}{\partial t^2} + \frac{\tau^2}{12} \frac{\partial^4 u}{\partial t^4} + O(\tau^4) \]

2.2 空间项加权展开

首先，空间中心差分算子的展开为： \[ \delta_x^2 u_j^n = \frac{\partial^2 u}{\partial x^2} + \frac{h^2}{12} \frac{\partial^4 u}{\partial x^4} + O(h^4) \]

接着处理加权平均项： \[ \theta \delta_x^2 u_j^{n-1} + (1-2\theta) \delta_x^2 u_j^n + \theta \delta_x^2 u_j^{n+1} = \delta_x^2 u_j^n + \theta \tau^2 \frac{\partial^2}{\partial t^2}(\delta_x^2 u_j^n) + O(\tau^4) \]

代入导数关系： \[ = \frac{\partial^2 u}{\partial x^2} + \frac{h^2}{12} \frac{\partial^4 u}{\partial x^4} + \theta \tau^2 \frac{\partial^4 u}{\partial t^2 \partial x^2} + O(\tau^4 + \tau^2 h^2) \]

3. 截断误差结果

利用原方程关系 \(\frac{\partial^4 u}{\partial t^4} = a^4 \frac{\partial^4 u}{\partial x^4}\) 以及 \(\frac{\partial^4 u}{\partial t^2 \partial x^2} = a^2 \frac{\partial^4 u}{\partial x^4}\)，整理得到：

\[ R_{ij}(h, \tau) = \left( \frac{\tau^2 a^4}{12} - \frac{a^2 h^2}{12} - \theta \tau^2 a^2 \right) \frac{\partial^4 u}{\partial x^4} + O(\tau^4 + \tau^2 h^2 + h^4) \]

Prob.2

5.1 特征分析

方程组 \(\frac{\partial \mathbf{u}}{\partial t} + \mathbf{A} \frac{\partial \mathbf{u}}{\partial x} = \mathbf{0}\) 的系数矩阵 \(\mathbf{A} = \begin{bmatrix} 0 & -1 \\ -1 & 0 \end{bmatrix}\) 具有特征值： \[\lambda_1 = 1, \quad \lambda_2 = -1\]

通过线性组合，可以定义特征变量： * \(w_1 = u - v\)：对应 \(\lambda_1 = 1\)，表示信息以速度 \(1\) 向右传播（由边界进入区域）。 * \(w_2 = u + v\)：对应 \(\lambda_2 = -1\)，表示信息以速度 \(-1\) 向左传播（由区域内部流向边界）。

5.2 为什么 \(v(0,t)\) 不能给边界条件？

从物理和数学逻辑上看，原因如下：

  1. 信息流向决定控制权：
    • 在左边界 \(x=0\) 处，只有一个特征方向（\(\lambda_1=1\)）是指向区域内部的。这意味着物理上只有一个“自由度”可以由外部干预。
    • 另一个特征方向（\(\lambda_2=-1\)）是从区域内部指向边界的。这意味着边界上 \(w_2\) 的值完全由 \(t=0\) 时刻的初始值经过演化后“携带”而来，外部无法强行规定。
  2. 变量间的耦合关系：
    • 既然 \(w_2(0,t) = u(0,t) + v(0,t)\) 的值已经由内部演化确定，而题目已经给定了 \(u(0,t) = 0\)，那么 \(v(0,t)\) 的值就必须满足 \(v(0,t) = w_2(0,t)\)。
    • 如果我们强行给 \(v(0,t)\) 设定一个值（例如 \(v(0,t)=g(t)\)），除非这个 \(g(t)\) 恰好等于内部传出来的 \(w_2(0,t)\)，否则会导致边界处的数学矛盾，使问题不适定 (Ill-posed)。

5.3 结论

对于双曲型方程组，在边界处能够给出的边界条件个数，必须等于指向区域内部的特征线数量。在本例中： * 正特征值个数 = 1 \(\rightarrow\) 只能给 1个 边界条件。 * 已知已给 \(u(0,t)=0\)，因此 \(v\) 在该边界上必须作为待求量，由系统内部决定。

实践作业

一、计算下面双曲型方程的近似解

\[ \begin{cases} \frac{\partial u}{\partial t} + \frac{\partial u}{\partial x} = 2t \sin x + t^2 \cos x, & 0 < t < 5, 0 < x < 10 \\ u|_{t=0} = 0 \\ u|_{x=0} = 0 \end{cases} \]

精确解：\(u = t^2 \sin x\)

要求：1. 选用显示格式计算，一阶、二阶格式各选一个，并选择合适步长验证稳定性，即有稳定和不稳定的算例。

  1. 至少选用一种隐式格式计算。

分析结果

计算参数

  • 时间步数 \(n = 250\)，空间步数 \(m = 1000\)
  • 时间步长 \(\Delta t = 5 / n = 2.0000 \times 10^{-2}\)
  • 空间步长 \(\Delta x = 10 / m = 1.0000 \times 10^{-2}\)
  • 对流速度 \(a = 1\)，CFL 数 \(= a \Delta t / \Delta x = 2.0000\)

显式格式（迎风 / Lax‑Wendroff）

CFL \(= 2 > 1\)，显式格式违反稳定性条件，计算结果发散：

  • \(u_{\min} = -2.5639 \times 10^{111}\)
  • \(u_{\max} = 2.5563 \times 10^{111}\)
  • \(\text{RMS}(u) = 1.9739 \times 10^{109}\)
  • \(\|e\|_{\infty} = 2.5639 \times 10^{111}\)
  • \(\|e\|_{\text{RMS}} = 1.9739 \times 10^{109}\)
  • 相对 \(\|e\|_{\infty} \approx 1.0338 \times 10^{110}\)
  • 相对 \(\text{RMS}(e) \approx 2.5691 \times 10^{108}\)

精确解 \(u = t^2 \sin x\)，显式格式因 CFL \(> 1\) 导致数值爆炸。

原因分析

  • 对于一阶线性对流方程 \(u_t + a u_x = 0\)（本问题右端项为源项，不影响稳定性分析的主部），显式迎风格式或 Lax‑Wendroff 格式的稳定性条件均为 CFL \(\le 1\)。
  • 当 CFL \(= 2 > 1\) 时，数值依赖域无法覆盖物理依赖域，时间推进过程中误差被指数放大，表现为解的振幅急剧增长（达到 \(10^{111}\) 量级），完全掩盖真实解。
  • 右端源项 \(2t\sin x + t^2\cos x\) 在格式不稳定时同样被放大，进一步加剧发散。

隐式格式（后向 Euler）

虽然 CFL \(= 2 > 1\)，但隐式格式无条件稳定，计算结果精确：

  • \(u_{\min} = -24.7221\)
  • \(u_{\max} = 24.7240\)
  • \(\text{RMS}(u) = 7.6602\)
  • \(\|e\|_{\infty} = 1.2658 \times 10^{-1}\)
  • \(\|e\|_{\text{RMS}} = 3.2766 \times 10^{-2}\)
  • 相对 \(\|e\|_{\infty} \approx 5.1038 \times 10^{-3}\)
  • 相对 \(\text{RMS}(e) \approx 4.2645 \times 10^{-3}\)

原因分析

  • 后向 Euler 格式是全隐式格式，其增长因子模长恒小于等于 1，对任意时间步长均无条件满足 von Neumann 稳定性条件。
  • 虽然 CFL \(= 2\) 远大于 \(1\)，但隐式格式通过求解全局线性方程组将当前时刻的信息与未来时刻耦合，避免了显式格式中时间步长受空间步长限制的问题。
  • 本算例中，后向 Euler 格式得到了与精确解高度吻合的数值解，相对误差仅为 \(10^{-3}\) 量级，验证了隐式格式处理大 CFL 数问题的优越性。

结论

  • 显式格式（迎风 / Lax‑Wendroff）在 CFL \(= 2\) 时严重不稳定，误差极大，原因是违反显式格式的必要稳定性条件。
  • 后向 Euler 隐式格式能有效克服 CFL 限制，获得高精度近似解，体现了隐式格式无条件稳定的核心优势。

二、计算下面双曲型方程的近似解

\[ \begin{cases} \frac{\partial^2 u}{\partial t^2} - \frac{\partial^2 u}{\partial x^2} = (t^2 - x^2) \sin(xt), & 0 < t < 1, 0 < x < 1 \\ u(x,0) = 0, \frac{\partial u}{\partial t}(x,0) = x, & 0 < x < 1 \\ u(0,t) = 0, u(1,t) = \sin t, & 0 < t \leq 1 \end{cases} \]

精确解：\(u = \sin(xt)\)

要求：1. 选用一种显式格式，并验证稳定性
2. 选用一种高阶精度的隐式格式计算。

第二题分析结果

计算参数（稳定算例 m1 / m2）

  • 时间步数 \(n = 60\)，空间步数 \(m = 60\)
  • 时间步长 \(\Delta t = 1.694915 \times 10^{-2}\)，空间步长 \(\Delta x = 1.694915 \times 10^{-2}\)
  • 波动方程 CFL 数 \(= \Delta t / \Delta x = 1.000000\)
  • 边界条件：\(u(0,t)=0\)，\(u(1,t)=\sin t\)；初始条件：\(u(x,0)=0\)，\(u_t(x,0)=x\)

中心差分格式 (Center Difference Scheme)

稳定算例（m1 / m2）

  • \(u_{\min} = 0\)，\(u_{\max} = 8.414710 \times 10^{-1}\)，\(\text{RMS}(u) \approx 3.161 \times 10^{-1}\)
  • 边界与初始条件残差均为 \(0\)（机器精度）
  • 最终时刻 \(t=1\) 处 \(\text{RMS}_x(u) \approx 5.2086 \times 10^{-1}\)
  • 离散能量均值约 \(2.7435 \times 10^{-1}\)，变化范围合理

数值解与精确解 \(u = \sin(xt)\) 高度吻合：在 \((x=1,t=1)\) 处精确值为 \(\sin 1 \approx 0.84147\)，数值解最大值即为该值。

不稳定算例（m3）

相同网格与 CFL 数，但计算发散：

  • \(u_{\min} \approx 1.6250 \times 10^{24}\)，\(u_{\max} \approx 1.6436 \times 10^{24}\)，\(\text{RMS}(u) \approx 9.2099 \times 10^{22}\)
  • 最终时刻 \(\text{RMS}_x(u) \approx 6.9834 \times 10^{23}\)
  • 离散能量均值高达 \(3.4103 \times 10^{48}\)，相对变化范围约 \(53.7\)

稳定性分析

  • 显式中心差分格式（对波动方程 \(u_{tt} - u_{xx} = f\) 采用二阶中心差分离散）的 von Neumann 稳定性条件为 CFL \(\le 1\)。
  • CFL \(= 1\) 处于临界稳定状态，理论上增长因子模为 \(1\)，无振幅放大。
  • 实际计算中，舍入误差、边界处理或源项离散的微小不对称可能激发虚假增长模式，导致长时间积分后出现爆炸性发散（如 m3）。
  • m1 / m2 在相同条件下保持稳定，说明稳定性对数值实现细节敏感；m3 可能是不同初始化或格式变体（如未正确处理初值导数）所致。

加权隐式格式 (Weighted Implicit Scheme)

采用隐式格式，网格加密至 \(n=m=80\)：

  • \(\Delta t = \Delta x = 1.265823 \times 10^{-2}\)，CFL \(= 1\)
  • \(u_{\min} = 0\)，\(u_{\max} = 8.414710 \times 10^{-1}\)，\(\text{RMS}(u) = 3.155968 \times 10^{-1}\)
  • 边界条件残差 \(\max|u(1,t)-\sin t| = 1.110223 \times 10^{-16}\)（机器精度）
  • 最终时刻 \(\text{RMS}_x(u) = 5.211956 \times 10^{-1}\)
  • 离散能量均值 \(2.746505 \times 10^{-1}\)，变化范围 \([1.65676,4.47499]\times10^{-1}\)

原因分析

  • 隐式格式（如 Crank‑Nicolson）对波动方程无条件稳定，允许使用较大时间步长而不产生数值爆炸。
  • 本例中即使 CFL \(= 1\)，隐式格式仍比显式格式具有更强的鲁棒性，且在更细网格上保持了与精确解的一致。
  • 相较于显式格式，隐式格式在每个时间步需要求解线性方程组，计算成本略高，但避免了临界稳定条件下的发散风险。
